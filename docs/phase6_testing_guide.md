## 功能测试

### 测试 1: 任务编辑对话框

**测试目的**: 验证任务编辑功能

**测试步骤**:
1. 启动应用
2. 右键点击角色 → 查看定时任务
3. 在任务列表中,点击"编辑"按钮
4. 修改任务名称
5. 修改 Cron 表达式(如 `0 9 * * *`)
6. 修改动作提示
7. 选择 Live2D 动作(可选)
8. 点击"保存"

**预期结果**:
- 编辑对话框打开
- 任务信息正确加载
- 修改后点击保存,任务更新成功
- 任务列表中显示更新后的信息

**验证点**:
- [ ] 对话框正常打开
- [ ] 所有字段正确显示
- [ ] 保存后任务信息已更新
- [ ] 任务列表刷新显示新信息
